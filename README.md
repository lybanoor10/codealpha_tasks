# 📊 CodeAlpha Data Analytics Internship Projects

## Overview

This repository contains the projects completed during my **Data Analytics Internship at CodeAlpha**. The internship focused on applying real-world data analytics techniques, starting from data collection through web scraping to data visualization and Natural Language Processing (NLP).

Each task demonstrates a different stage of the data analytics workflow, including data acquisition, cleaning, exploratory analysis, visualization, and sentiment analysis.

---

# Internship Objectives

* Collect data from public websites using Python.
* Clean and prepare datasets for analysis.
* Perform Exploratory Data Analysis (EDA).
* Create professional data visualizations.
* Analyze text data using Natural Language Processing (NLP).
* Generate actionable insights to support decision-making.

# Task 1: Web Scraping

### Objective

Collect structured data from a public website using Python.

### Tasks Performed

* Sent HTTP requests
* Parsed HTML using BeautifulSoup
* Navigated through 50 web pages
* Extracted:

  * Book Title
  * Price
  * Availability
  * Rating
* Created a structured CSV dataset
* Performed initial data cleaning

### Skills Demonstrated

* Web Scraping
* HTML Parsing
* Pagination
* Data Collection
* Dataset Creation

### Output Files

* `scrape_all_pages.py`
* `data_cleaning.py`
* `all_books.csv`
* `clean_books.csv`
